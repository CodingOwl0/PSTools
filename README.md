# PSTools / CMDTools
<h2>Simple Tools for Powershell and CMD (Command-Prompt)</h2><br>

<h3>Overview:</h3><br>

<b>CMDCrypt.exe</b><br>
Encrypt and decrypt files (and strings) by using AES CBC Algorythm.<br>

Usage: <br>
cmdCrypt.exe <Parameter> "Source(-Filename)" "Output(-Filename)" "Password" "Salt" <br>
-f  : Encrypt a file   -s  : Encrypt a string     -d  : Encrypt whole directory <br>
-fd : Decrypt a file   -sd : Decrypt a string     -dd : Decrypt whole directory <br>

<b>Compare.exe</b><br>
Compare the files in a directory with another.<br>

Usage:<br>
compare.exe Folder1 Folder2 <br>

<b>TBL.exe</b><br>
Shows you a (very) simple table view of your csv-file.<br>

Usage:<br>
 tbl (filepath) <seperator><br>
